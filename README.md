# flange-couple-four-cylinder-engine

This repository contains the completed tasks submitted for the CodeAlpha AutoCAD Internship using SolidWorks.

Task 2 – Flange Coupling Mechanical Drawing

A complete 2D technical drawing of a flange coupling was designed using SolidWorks.
The workflow included:

- Creating the 3D flange coupling model
- Generating engineering drawing sheets
- Creating Front View, Top View, and Side View
- Applying proper dimensions and annotations
- Following standard mechanical drafting practices
- Exporting the final drawing as PDF and source files

This task demonstrates technical drawing skills and understanding of mechanical design standards.
